---
title: "Create Event"
api: "POST https://api.highstorm.com/v1/events/{channel}"
description: "This endpoint creates a new event"
---


### Header



<ParamField header="Authorization" type="string" required placeholder="Bearer api_kRKeD7pzNVsX7maoCmCng">
  The authorization token. You can find this in your [account settings](https://app.highstorm.com/home).
</ParamField>



<ParamField header="Content-Type" type="string" required placeholder="application/json">
  The content type of the request. This must be `application/json`.

</ParamField>



### Path

<ParamField path="channel" type="string" required>
   The channel name where the event will be ingested.
   Channel names must be alphanumeric and can contain underscores, dashes, and periods.
</ParamField>

### Body

<ParamField body="event" type="string" required>
  A short description of what happened.
</ParamField>

<ParamField body="content" type="string" optional >
 The content of the event. This can be as long as you want.
 </ParamField>

<ParamField body="metadata" type="key-value" optional >
        Key value data, you can use this to store any additional information.
 </ParamField>

<ParamField body="time" type="integer" optional default="now">
        Timestamp of when the event was created (Unix timestamp in milliseconds). If not provided, the current time will be used.
 </ParamField>

### Response

<ResponseField name="id" type="string">
  The ID of the event.
</ResponseField>

<RequestExample>

```bash curl
curl -XPOST 'https://api.highstorm.app/v1/events/user.signup' \
--header 'Content-Type: application/json' \
--header 'Authorization: Bearer <token>' \
--data-raw '{
    "event": "Chronark signed up",
    "metadata": {
        "user_id": "123",
        "source": "Twitter"
    },
}'
```

```ts fetch
fetch('https://api.highstorm.app/v1/events/user.signup', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json',
    'Authorization': 'Bearer <token>'
  },
  body: JSON.stringify({
    event: 'Chronark signed up',
    metadata: {
      user_id: '123',
      source: 'Twitter'
    }
  })
})

```

</RequestExample>

<ResponseExample>

```json Response
{
  "id": "evt_HMyvFDEhzwvAPVfsYqZNQ"
}
```

</ResponseExample>
